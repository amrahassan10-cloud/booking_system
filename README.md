# booking_system

A Java-based booking system with a graphical user interface (GUI) that allows users to create, manage, and cancel bookings. The application is designed to connect to any user-provided database for data storage.

🎮 Features

User-friendly GUI
Create, update, and cancel bookings
Connects to a custom database (user-configured)
Input validation and error handling

Technologies Used

Java
GUI: (Swing / JavaFX)
Database: User-defined (e.g., MySQL, SQLite)

▶️ How to Run

Set up your own database
Configure the database connection in the project settings
Run the application from your IDE
